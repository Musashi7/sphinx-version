# What's new

## v5.0

Full PSJ functions are introduced, with below features:

- Add PSJ-Command

- Add more functions to PSJ-GUI

- Add more functions for PSJ-Utility

- Add 3 tutorials pages for PSJ

## v4.1.4

First introduce PSJ which is the improvement of traditional Jupiter's macro:

- Add more functions to macro

- Improve IDE: highlight, auto completion, tooltip, user-defined function tooltip

- Add more functions for PSJ-Utility (former name of API)

- Introduce a GUI Command Builder with built in API, for creating UI components inside Jupiter with just drag-and-drop operations

- Introduce new Python API window, help user debug script easier

- Add tutorials pages for PSJ
