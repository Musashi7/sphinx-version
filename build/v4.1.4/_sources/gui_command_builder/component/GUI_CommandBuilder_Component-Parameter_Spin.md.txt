# Spin

Create a spin button.  

![](./../../_images/GUI_CommandBuilder_Component-Parameter_Spin.png)  
  
Preview dialog:

![](./../../_images/GUI_CommandBuilder_Component-Parameter_Spin2.png)  
  

**Name**

Enter the name of the spin component.

**Min**

Specify the minimum value of the spin range.

**Max**

Specify the maximum value of the spin range.

**Value**

Specify an initial value of the spin range.

**Enable**

Specify whether to enable it or not.

- **Check On** : Available
- **Check Off** : Unavailable (grayed out)
