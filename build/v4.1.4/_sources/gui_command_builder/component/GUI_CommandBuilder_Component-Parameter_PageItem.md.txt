# Page Item

Create a page inside the page container (used for wizard). 

![](./../../_images/GUI_CommandBuilder_Component-Parameter_PageItem.png)  
  
 Preview dialog:

![](./../../_images/GUI_CommandBuilder_Component-Parameter_PageCtrl2.png)  

 ※Page Item is used inside a Page Control.

## Parameters

**Name**

Enter the name of the Page Item component.

**Text**

Specify the header of the Page Item.
