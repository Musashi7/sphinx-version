# Image Control

Create an image container.  

![](./../../_images/GUI_CommandBuilder_Component-Parameter_ImageCtrl.png)  

Preview dialog:

![](./../../_images/GUI_CommandBuilder_Component-Parameter_ImageCtrl2.png)  
  
## Parameters

**Name**

   Enter the name of the image control component.

**Image File**

   Display the link to the image.

## Command to use

| Command                                                     | Description                                                  | Sample Code                                                  |
| ----------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| dlg.set_image_file("name", "link")                         | Set the location to the image file  | *Input:*<br>`dlg.set_image_file("ImageCtrl", "C:/Users/M3.JPG")`    |
