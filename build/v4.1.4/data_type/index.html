Data Type¶

Basic data type¶

Basic data type used in Jupiter macro/API.

*All Python data types can be used when using the Python API.

Jupiter Macro/API data type

Abbreviation

Description

Integer

int

Represents a positive or negative value with no decimal point.
2147483647 means the default value (blank=no input) in the integer value field

Boolean

bool

Represents the boolean values False(0) and True(1)

Float

float

Represents a computer level double precision floating point number.
1.79769e+308 means the default value (Blank=no input) in the real number field

String

str

Represents a string. Enclose with ""

Cursor

cursor

TypeID: Represents an entity in the form of ID.
Example:
3:1
[3:1,3:2]
