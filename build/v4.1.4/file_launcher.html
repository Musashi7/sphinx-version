File Launcher¶

Interface¶

Buton

Description

S

Register the Python file

C

Clear the registered Python file

R

Replace the registered Python file with another Python file

Func

Execute the registered Python program

Operating procedure¶

  1. Click the S button and select the Python file

  2. When you select a file, the green mark to the left of the Function button turns red

  3. Click the Function button to execute the program
