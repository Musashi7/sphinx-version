# Label

Add a label.  

![](./../../_images/GUI_CommandBuilder_Component-Parameter_Label.png)  
  
Preview dialog:

![](./../../_images/GUI_CommandBuilder_Component-Parameter_Label2.png)  
  
## Parameters

**Text**

Enter the text displayed as a label.

**Name**

Enter the name of the label component.

**Enable**

Specify whether to enable it or not.

- **Check On** : Available
- **Check Off** : Unavailable (grayed out)

**Width**

Specify the minimum width of the label. (in pixels)

**Height**

Specify the minimum height of the label. (in pixels)
