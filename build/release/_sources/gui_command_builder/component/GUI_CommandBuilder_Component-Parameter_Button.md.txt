## Parameters

**Text**

Enter the text appears on the button.

**Name**

Enter the name of the button component.

**Enable**

Specify whether to enable it or not.

- **Check On** : Available
- **Check Off** : Unavailable (grayed out)

**Width**

Specify the minimum width of the button. (in pixels)

**Height**

Specify the minimum height of the button. (in pixels)
